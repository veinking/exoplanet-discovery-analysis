# Exoplanet Discovery Analysis

This project analyzes confirmed exoplanets using publicly available NASA Kepler mission data to explore planetary characteristics and discovery patterns.

The analysis focuses on three key dimensions of exoplanet discovery:

• Planetary size distribution  
• Stellar energy exposure  
• Discovery confidence scores  

Using Python and Pandas, the dataset is cleaned and filtered to include only confirmed planets. Features such as planetary radius, stellar energy received, and detection confidence are analyzed to identify patterns across different categories of planets.

The resulting datasets are exported for use in Tableau to support data visualization and interactive dashboards.
